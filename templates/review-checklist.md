# 审稿检查清单

## 1. 去AI味
- [ ] 无"值得注意的是"等套话
- [ ] 无"不仅...而且..."等书面句式
- [ ] 读出声来像真人在说话
- [ ] 有口语化表达和不完美句式

## 2. 结构完整性
- [ ] Hook 前3秒有钩子
- [ ] 每30秒有新信息或转折
- [ ] 论据具体（有数据/案例/引用）
- [ ] 情绪有起伏
- [ ] 结尾有力，有金句
- [ ] 画面指引已标注

## 3. 定位一致性
- [ ] 观点符合账号定位
- [ ] 用词风格一致
- [ ] 目标受众能听懂

## 4. 信息质量
- [ ] 信息密度足够（每分钟有足够干货）
- [ ] 没有空洞的口号
- [ ] 引用的数据/案例可靠
