# 去AI味检查清单

## 必须删除的词/句式

| 类型 | 示例 | 替换为 |
|------|------|--------|
| 过渡套话 | "值得注意的是" | 直接说重点 |
| 总结套话 | "总的来说"、"综上所述" | 删掉，直接给结论 |
| 并列套话 | "不仅...而且..."、"一方面...另一方面..." | 拆成两句短句 |
| 引导套话 | "让我们一起来看看"、"接下来" | 删掉 |
| 程度套话 | "非常重要"、"极其关键" | 用具体事实代替 |
| 假谦虚 | "在我看来"、"我个人认为" | 直接说观点 |
| 列举套话 | "首先...其次...最后..." | 打乱顺序，用口语连接 |

## 必须调整的风格

| 书面感 | 口语感 |
|--------|--------|
| 然而 | 但是 / 但 |
| 因此 | 所以 |
| 此外 | 还有一个事 |
| 目前 | 现在 |
| 进行 | 做 / 搞 |
| 实现 | 做到 |
| 针对 | 对 / 关于 |
| 显著 | 明显 |
| 至关重要 | 特别重要 / 没这个就完了 |

## 检查方法

**读出声测试：** 把稿子读出声来。如果某句话你日常绝对不会这么说，就要改。

**朋友聊天测试：** 想象你在跟朋友聊这个事。你会怎么说？用那个方式写。

## 允许保留的"不完美"

- 口头禅（"你想想看"、"对吧"）
- 重复强调（"真的，真的很猛"）
- 不完整句（"然后呢？没了。就这样。"）
- 感叹词（"卧槽"、"我靠"、"离谱"）

这些"不完美"恰恰是人味的来源。
