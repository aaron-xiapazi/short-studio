# 标题公式库

视频标题决定封面点击率。以下是验证有效的标题模式。

## 核心公式

### 1. 反常识
```
[大家都以为的] + 其实 + [真相]
```
例："所有人都在夸AI写作，但用过的人都知道这个坑"

### 2. 数字冲击
```
[具体数字] + [动作] + [结果]
```
例："我用3天做了一个AI工具，第一周赚了2万"

### 3. 悬念
```
[现象/发现] + [暗示惊人结论]
```
例："我研究了100个爆款视频，发现它们都有一个共同点"

### 4. 对比冲突
```
[A] vs [B]，[出人意料的结论]
```
例："ChatGPT和Claude打了一架，赢的那个让我意外"

### 5. 利益直给
```
[动作/方法] + [受众想要的结果]
```
例："用这个方法，你的视频完播率能翻倍"

### 6. 身份共鸣
```
[身份标签] + [痛点/场景]
```
例："做自媒体第3年，我终于想通了一件事"

## 标题禁忌

- 不要用问号结尾的标题（"你知道吗？"——不知道，也不想知道）
- 不要用"干货"、"建议收藏"这种词（被用烂了）
- 不要超过20个字（太长没人看完）
- 不要跟观点无关的标题党（点进来发现不是那回事会被骂）

## 使用方式

选题确定后，用以上公式生成 3-5 个标题候选，让用户选择。
