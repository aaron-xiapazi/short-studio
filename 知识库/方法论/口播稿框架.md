# 口播稿框架

## 核心原则

口播稿是**说出来**的，不是写出来的。写的时候脑子里要有一个人在对面听你讲。

## 结构框架（3-5分钟视频）

### 1. Hook（0-3秒）
**任务：** 让人停下来。

Hook 公式：
- **反常识：** "所有人都在说X，但真相是Y"
- **悬念：** "我发现了一个东西，可能会改变你对X的看法"
- **冲突：** "X和Y打起来了，赢的那个你绝对想不到"
- **利益：** "用了这个方法，我X天做到了Y"

### 2. 背景铺垫（3-30秒）
**任务：** 让人知道你要讲什么，为什么要听。

- 用一句话交代背景
- 快速建立"这事跟你有关"的感觉
- 不要铺太长，30秒内必须进入正题

### 3. 核心内容（30秒-3分钟）
**任务：** 输出干货。

**节奏要求：每30秒一个钩子**
- 新信息、新案例、反转、数据、金句——每半分钟至少出现一个
- 如果超过30秒没有新东西，观众就走了

**论据要求：具体、具体、再具体**
- ❌ "效果很好" → ✅ "3天涨了5万播放"
- ❌ "很多人在用" → ✅ "我认识的10个创作者里有8个在用"

### 4. 深层洞察（3分-4分钟）
**任务：** 拔高一层，给出别人没说过的观点。

- 不只是"是什么"，更是"为什么"和"意味着什么"
- 这里是你跟其他账号拉开差距的地方

### 5. 收尾 + CTA（最后30秒）
**任务：** 留下印象，引导互动。

- 金句收尾（让人想截图/转发）
- CTA 不要太硬："你觉得呢？评论区聊聊"
- 不要用"关注我获取更多"这种废话

## 字数参考

| 时长 | 字数 |
|------|------|
| 1分钟 | 250-300字 |
| 3分钟 | 750-900字 |
| 5分钟 | 1200-1500字 |

## 禁忌

- 不要用"大家好我是xxx"开头（浪费3秒）
- 不要用"今天我们来聊聊"（没有信息）
- 不要在结尾总结全文（观众已经看了，不需要你重复）
- 不要堆砌没有支撑的观点（空喊口号）
