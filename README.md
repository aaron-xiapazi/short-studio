# Short Studio

> AI 内容工厂：从定位到发布的视频内容生产全链路。

## 这是什么？

Short Studio 是一个基于 [飞书 CLI](https://github.com/larksuite/cli) 的 Skill 技能包。它不是一个跑一次就完事的工具，而是一个**有状态、会学习、持续进化的内容工厂**。

你给它投喂定位、方法论、对标账号，它帮你跑完选题→文案→审稿→发布的全流程。用得越多，它越了解你，产出质量越高。

## 工厂流程

```
初始化（投喂定位 + 方法论 + 对标账号）
       ↓
   ┌───────┐      ┌───────┐      ┌───────┐      ┌───────┐
   │ 选题   │ ──▶ │ 文案   │ ──▶ │ 审稿   │ ──▶ │ 发布   │
   │ topic  │      │ script │      │ review │      │publish │
   └───────┘      └───────┘      └───────┘      └───────┘
       ↑                                              │
       └──────────── 效果数据回流 ◀────────────────────┘
```

每个环节独立可用，也可以串联运行。发布后的数据会回流到选题环节，形成闭环。

## Skills

| 命令 | 功能 | 状态 |
|------|------|------|
| `/studio` | 主入口，识别意图，路由到子 skill | ✅ |
| `/studio-init` | 初始化：投喂账号定位、个人定位、平台账号、对标账号 | ✅ |
| `/studio-topic` | 选题：热点抓取 × 定位匹配 × 历史数据 | 🚧 |
| `/studio-script` | 文案：口播稿写作（hook + 结构 + 论据 + 情绪曲线） | 🚧 |
| `/studio-review` | 审稿：自检 + 去AI味 + 方法论校验 | 🚧 |
| `/studio-scrape` | 采集：对标账号内容抓取，喂入知识库 | 🚧 |
| `/studio-publish` | 发布：多平台推送（v2） | 📋 |

## 快速开始

### 1. 安装飞书 CLI

```bash
npm install -g @larksuite/cli
lark-cli config init
```

### 2. 安装 Short Studio

```bash
git clone https://github.com/aaron-xiapazi/short-studio.git ~/.claude/skills/short-studio
cd ~/.claude/skills/short-studio && bash setup
```

### 3. 初始化工厂

```
/studio-init
```

首次使用会引导你完成：
- 账号定位（这个账号做什么方向）
- 个人定位（你是谁，差异化在哪）
- 平台账号（抖音/小红书/B站的主页地址）
- 对标账号（你想参考谁）

初始化完成后，工厂就知道"我是谁、为谁做内容"。

### 4. 开始生产

```
/studio          # 进入工厂，告诉它你想干嘛
/studio-topic    # 直接找选题
/studio-script   # 直接写文案
```

## 知识架构

工厂的核心竞争力在于**知识积累**，分三层：

```
┌─────────────────────────────┐
│  SKILL.md（执行指令）         │  AI 知道该怎么做
├─────────────────────────────┤
│  方法论（框架）               │  选题矩阵、口播稿框架、去AI味清单
├─────────────────────────────┤
│  原子库 + 自有库（数据）      │  对标内容 + 自己的效果数据
└─────────────────────────────┘
```

- **方法论**是你的内容方法论沉淀，可以不断打磨
- **原子库**通过 `/studio-scrape` 从对标账号采集，持续增长
- **自有库**记录你自己发布内容的效果数据，形成反馈闭环

## 共享工具层

Skills 背后有一层共享工具，封装了抖音采集、视频转文案等能力：

| 工具 | 能力 | 消耗 |
|------|------|------|
| `douyin-profile` | 抖音主页采集 | 1 credit |
| `video2script` | 视频转文案（ASR） | 2 credits |
| `douyin-search` | 关键词搜索 | 1 credit |
| `trending` | 热点趋势 | 1 credit |
| `comments` | 抓评论区 | 1 credit |
| `web-search` | 通用搜索（博查等） | 1 credit |

- **零配置**：你不需要自己申请任何 API Key，我们统一提供
- **免费试用**：新用户自带试用额度
- **离线可用**：工具不可用时，工厂降级为手动模式，照样能跑

→ 详见 [tools/TOOLS.md](tools/TOOLS.md)
